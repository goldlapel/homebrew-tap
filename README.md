# homebrew-tap

Homebrew tap for [Gold Lapel](https://goldlapel.com) — a self-optimizing Postgres proxy.

Product site: https://goldlapel.com · Docs: https://goldlapel.com/docs

## Install (users)

```bash
brew install goldlapel/tap/goldlapel
```

## Repo structure

- `Formula/goldlapel.rb` — the formula (URLs + SHA256s per platform, auto-generated by the main repo's release pipeline)

## Updating the formula

The formula is regenerated automatically by `release-homebrew.yml` in the main `goldlapel/goldlapel` repo on every tagged release. Do not edit `Formula/goldlapel.rb` by hand — it will be overwritten.
